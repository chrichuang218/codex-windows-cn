# v1 is a Chinese install and update assistant

v1 will ship as a Chinese install and update assistant: it preserves the five main user-visible paths from the reference project: install, proxy launch, check/update, uninstall, and launcher self-update. Maintaining the Windows integration required by the current installation, including shortcuts and the `codex://` session-link registration, is part of those install and update paths. General diagnostics, system repair flows, network checks, and other toolbox-style enhancements are intentionally deferred so the first release can prove the core experience without expanding product scope.
