# Codex Windows CN

This context defines the product language for Codex Windows 中文助手. It keeps product terms stable while the implementation evolves.

## Language

**Codex Windows 中文助手**:
面向中文 Windows 用户的 Codex 桌面应用安装、更新、启动与卸载助手。它不是 OpenAI 官方项目，也不是 Codex 应用本体。
_Avoid_: Codex 官方安装器, Codex 中文版

**中文安装更新助手**:
v1 的产品边界，指以中文体验交付现有安装、更新、启动、卸载和自更新闭环。它不包含诊断、修复、网络优化等增强能力。
_Avoid_: 中文工具箱, 诊断修复平台

**功能不变**:
v1 的范围约束，指保持参考项目已经具备的用户可见能力与行为边界，不在首版增加新的产品能力。
_Avoid_: 重写功能, 顺手增强

**五条主路径**:
v1 功能不变的验收口径，指安装、代理启动、检查更新/更新、卸载、自更新这五条用户可见路径。
_Avoid_: 全量功能矩阵, 后续增强清单

**后续优化**:
v1 之后的迭代空间，包含诊断报告、一键修复、网络检测、代理说明和中文问题排查等增强能力。
_Avoid_: v1 必做项, 首版范围

**本机已安装版本**:
已经完整解压到 `versions/` 并通过启动入口校验、可直接运行的官方应用版本。它不等同于下载缓存中的 MSIX 文件。
_Avoid_: 下载版本, 历史安装包

**历史启动**:
用户显式选择并启动某个非最新的本机已安装版本。未选择时仍启动本机最新版本。
_Avoid_: 历史版本下载, 自动降级

**版本保留策略**:
控制更新完成后是否自动清理旧的本机已安装版本。策略可以按数量保留最近版本，也可以全部保留并仅允许用户手动删除。
_Avoid_: 下载缓存数量, 强制清理
